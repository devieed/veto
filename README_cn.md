# 足球反波胆系统

*一个去中心化的平台，用于参与足球比赛的反波胆投注。*

## 概述

本项目提供了一个平台，允许用户参与足球比赛的反波胆投注。该系统利用区块链技术和智能合约来确保透明度和公平性。它使用 Java (JDK 21) 构建，并使用 Alchemy 进行支付通知。

## 主要功能

*   **反波胆投注:** 允许用户投注反对比赛结果。
*   **ETH-USDT 支持:** 目前支持 ETH-USDT 交易对。
*   **Alchemy 集成:** 使用 Alchemy 进行安全的支付回调。
*   **Redis 存储:** 钱包信息安全地存储在 Redis 中。
*   **管理面板:** 专用的管理面板，用于系统配置和管理。

## 前置条件

*   已安装 Docker 和 Docker Compose。
*   已安装 Java JDK 21。
*   拥有 Alchemy 账户和 API 密钥。
*   Redis 服务器正在运行。

## 设置与部署

1.  **启动基础服务:** 导航到 `.docker` 目录并运行：
    ```bash
    docker compose up
    ```
2.  **管理面板设置:**
    *   启动管理面板应用程序。
    *   默认用户名：`admin`
    *   默认密码：`admin`
    *   在管理面板中配置 Alchemy API 密钥。
3.  **启用系统:** 在管理面板中的系统设置中，确保将“系统启用”配置为 `true`。
4.  **部署 `veto-boot` 服务:** 部署 `veto-boot` 服务。
5.  **静态页面部署:** 编译静态页面并将它们挂载到指定的容器（参考容器仓库：`xxx`）。

## 使用方法

1.  **选择 ETH-USDT:** 系统运行后，请确保在系统设置中选择了 ETH-USDT 货币对。
2.  **钱包管理:** 钱包信息会自动管理并存储在 Redis 中。
3.  **反波胆投注:** 用户可以通过前端界面参与反波胆投注。
