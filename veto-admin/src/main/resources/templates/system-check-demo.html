系统启动检查

系统检测到以下核心参数尚未配置，请完成配置以确保系统正常运行：

缺失的配置项：

  • 系统名称 (veto_config_system_name)
  • 系统域名 (veto_config_system_domain)
  • Token密钥 (veto_config_token_secret_key)
  • 系统钱包地址 (veto_config_system_wallet)
  • 系统币种类型 (veto_config_system_coin_type)
请确保所有必填项都已正确填写，这些参数是系统正常运行的基础保障。
